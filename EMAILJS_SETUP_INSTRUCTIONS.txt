EMAILJS SETUP INSTRUCTIONS
==========================

1. LOG IN TO EMAILJS
   - Go to https://www.emailjs.com/
   - Sign in to your account (or create one if you don't have it)

2. CREATE EMAIL TEMPLATES

   For Contact Form:
   a. Go to "Email Templates" section
   b. Click "Create new template"
   c. Set template name to "Contact Form Template"
   d. Set template ID to "template_uvnmczx" (this matches your current code)
   e. Copy the content from "emailjs_template.html" file and paste it into the template editor
   f. Replace the placeholder content with the EmailJS template variables:
      - {{name}} for the name field
      - {{email}} for the email field
      - {{phone}} for the phone field
      - {{subject}} for the subject field
      - {{message}} for the message field

   For Service Request Forms:
   a. Go to "Email Templates" section
   b. Click "Create new template"
   c. Set template name to "Service Request Template"
   d. You can use the same template ID or create a new one
   e. Copy the content from "emailjs_service_template.html" file and paste it into the template editor
   f. Replace the placeholder content with the EmailJS template variables:
      - {{name}} for the name field
      - {{email}} for the email field
      - {{phone}} for the phone field
      - {{service}} for the service field
      - {{message}} for the message field

3. CONFIGURE EMAIL SERVICE
